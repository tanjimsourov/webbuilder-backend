# provider/

Pluggable infrastructure provider abstractions.

- Storage provider abstraction.
- Image transformation URL provider.
- Email sending provider.
- Search indexing provider.
- DNS/domain verification provider.
- Payment provider abstraction (`StripePaymentProvider` default).
- Shipping rate provider abstraction (`InternalShippingProvider` default).
- Tax calculation provider abstraction (`InternalTaxProvider` default).
